Verfügbare Funktionen:

    Englische Funktionsnamen:

  • move(): Bewegt den Spieler einen Schritt in die aktuelle Richtung.
  • turnLeft(): Dreht den Spieler um 90° nach links.
  • noWater(): Prüft, ob vor dem Spieler Land (kein Wasser) ist. true, wenn Land vor dem Spieler ist, sonst false.
  • setMarker(): Setzt eine Markierung an der aktuellen Position des Spielers.
  • onTreasure(): Prüft, ob der Spieler auf dem Schatz steht. true, wenn auf dem Schatz, sonst false.
  • onMarker(): Prüft, ob der Spieler auf einer Markierung steht. true, wenn auf Markierung, sonst false.

    • Deutsche Funktionsnamen:

  • vor(): Bewegt den Spieler einen Schritt in die aktuelle Richtung.
  • links(): Dreht den Spieler um 90° nach links.
  • vorneFrei(): Prüft, ob vor dem Spieler Land (kein Wasser) ist. true, wenn Land vor dem Spieler ist, sonst false.
  • setzteMarkierung(): Setzt eine Markierung an der aktuellen Position des Spielers.
  • aufSchatz(): Prüft, ob der Spieler auf dem Schatz steht. true, wenn auf dem Schatz, sonst false.
  • aufMarkierung(): Prüft, ob der Spieler auf einer Markierung steht. true, wenn auf Markierung, sonst false.

Hinweis: Funktionen müssen mit der Syntax function functionName() {} deklariert werden.
