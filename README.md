# Vite Projekt

Dieses Projekt wurde mit [Vite](https://vitejs.dev/) erstellt.

## 🚀 Installation & Start

### 1. Repository klonen
```sh
git clone <repository-url>
cd <projektname>
```

### 2. Abhängigkeiten installieren
```sh
npm install
```

### 3. Entwicklung starten
```sh
npm run dev
```

## 📦 Produktion bauen
```sh
npm run build
```
